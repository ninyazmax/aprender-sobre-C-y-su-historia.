Aprendiendo C++ desde Cero

"El código es poesía lógica que da vida a las ideas." - Steven Gabriel

¿Qué es C++?

C++ es un lenguaje de programación poderoso y versátil, ideal para quienes quieren entender cómo funciona una computadora desde adentro. Aquí aprenderás paso a paso, con ejemplos y ejercicios interactivos, todo lo necesario para comenzar tu viaje como programador.

¿Por qué aprender C++?
  • Es uno de los lenguajes más usados en la industria.
  • Te da una base sólida para aprender otros lenguajes.
  • Permite crear desde videojuegos hasta sistemas operativos.

Tu Primer Programa en C++

¡Vamos a escribir tu primer programa! El clásico Hola Mundo:

¿Qué hace este código?
  • Incluye la biblioteca iostream para mostrar texto en pantalla.
  • Define la función principal main() donde inicia el programa.
  • Usa std::cout para imprimir el mensaje "¡Hola Mundo!" en la consola.
  • return 0; indica que el programa terminó correctamente.
// Mi primer programa en C++
#include <iostream>

int main() {
    std::cout << "¡Hola Mundo!" << std::endl;
    return 0;
}

Variables y Tipos de Datos

Las variables son como cajas donde guardamos información. En C++, cada variable debe tener un tipo.

¿Cómo se usan?
  • Primero declaras el tipo de dato (int, float, etc).
  • Luego le das un nombre a la variable y puedes asignarle un valor.
  • Por ejemplo: int edad = 18; crea una variable llamada edad y le asigna el valor 18.
Tipo¿Para qué sirve?Ejemplo
intNúmeros enteros 1, 2 etcint edad = 18;
floatNúmeros decimales simples 1.1, 1.2, 1.3, etcfloat altura = 1.75f;
doubleNúmeros decimales más precisosdouble pi = 3.1416;
charUn solo carácter (el cual consiste en basicamente una sola letra, numero o simbolo)char letra = 'A';

Consejo: Usa nombres de variables claros y descriptivos, ¡te ahorrarás dolores de cabeza! (aunque también puedes optar por usar nombres cortos si estás haciendo un programa pequeño o de prueba)

Operadores en C++

Los operadores permiten hacer operaciones matemáticas y lógicas.

¿Cómo funciona el ejemplo?
  • Declara dos variables enteras a y b con valores 10 y 3.
  • Realiza operaciones básicas: suma, resta, multiplicación, división y módulo.
  • Cada resultado se guarda en una variable diferente.
int a = 10, b = 3;
int suma = a + b;    // Suma (13)
int resta = a - b;   // Resta (7)
int producto = a * b; // Multiplicación (30)
int division = a / b; // División entera (3)
int modulo = a % b;  // Módulo (resto de la división, 1)
Tip: Usa paréntesis para asegurarte del orden de las operaciones.

Toma de Decisiones

Los programas pueden tomar decisiones usando condiciones (condicionales). En C++ usamos if, else if y else.

¿Qué hace este código?
  • Pide al usuario que ingrese su edad usando std::cin.
  • Si la edad es mayor o igual a 18, muestra "Eres mayor de edad".
  • Si no, muestra "Eres menor de edad".
int edad;
std::cin >> edad;
if (edad >= 18) {
    std::cout << "Eres mayor de edad";
} else {
    std::cout << "Eres menor de edad";
}
Consejo: Prueba siempre los casos límite, por ejemplo, 17, 18 y 19.

Ciclos: Repetir sin cansarse

Los ciclos permiten repetir instrucciones varias veces. Los más usados son for y while.

¿Qué hace el ciclo for?
  • Declara una variable i que empieza en 0.
  • Mientras i sea menor que 5, ejecuta el bloque de código.
  • En cada vuelta, muestra el valor de i y lo incrementa en 1.
for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}
¿Y el ciclo while?
  • Declara una variable contador en 0.
  • Mientras contador sea menor que 3, muestra su valor y lo incrementa.
int contador = 0;
while (contador < 3) {
    std::cout << contador << std::endl;
    contador++;
}

Funciones: Reutiliza tu código

Una función es un bloque de código que realiza una tarea específica. Ayuda a organizar y reutilizar el código.

¿Cómo funciona el ejemplo?
  • La función sumar recibe dos enteros y devuelve su suma.
  • La función factorial es recursiva: se llama a sí misma para calcular el factorial de un número.
  • En main() se muestran ejemplos de uso de ambas funciones.
int sumar(int a, int b) {
    return a + b;
}

// Función recursiva para calcular el factorial de un número
int factorial(int n) {
    if (n <= 1) return 1;
    else return n * factorial(n - 1);
}

int main() {
    std::cout << sumar(3, 4) << std::endl; // Imprime 7
    std::cout << factorial(5) << std::endl; // Imprime 120
    return 0;
}
¿Qué es recursividad?
  • Una función recursiva se llama a sí misma para resolver problemas que pueden dividirse en subproblemas más pequeños.
  • En este ejemplo, factorial(5) calcula 5 × 4 × 3 × 2 × 1.
  • ¡Cuidado! Toda función recursiva debe tener una condición de parada para evitar bucles infinitos.

Arreglos en C++

Un arreglo es una colección de variables del mismo tipo. Imagina una fila de casilleros donde cada uno guarda un valor.

¿Cómo funciona el ejemplo?
  • Declara un arreglo de 5 enteros llamado numeros y le asigna los valores 1, 2, 3, 4 y 5.
  • numeros[0] accede al primer elemento del arreglo (el número 1).
  • Recuerda: los arreglos empiezan en el índice 0.
int numeros[5] = {1, 2, 3, 4, 5};
std::cout << numeros[0]; // Imprime 1
Recuerda: El primer elemento es el índice 0.

Punteros: ¿Qué son y para qué sirven?

Un puntero es una variable especial que almacena la dirección de memoria de otra variable. Es decir, en vez de guardar un valor directamente, guarda la ubicación donde está ese valor en la memoria del computador.
Los punteros son fundamentales en C++ para manipular datos de forma eficiente, crear estructuras dinámicas (como listas enlazadas) y para pasar grandes cantidades de datos a funciones sin copiarlos.

¿Cómo funciona el ejemplo?
  • int x = 10; crea una variable entera llamada x y le asigna el valor 10.
  • int* p = &x; declara un puntero a entero llamado p y lo inicializa con la dirección de memoria de x (el operador & obtiene la dirección de una variable).
  • std::cout << p; imprime la dirección de memoria almacenada en p (por ejemplo, 0x7ffee4b8c8ac).
  • *p es el operador de desreferenciación: accede al valor almacenado en la dirección a la que apunta p. Así, *p es igual a x (10).
int x = 10;
int* p = &x; // p guarda la dirección de x
std::cout << "Dirección de x: " << p << std::endl; // Muestra la dirección de x
std::cout << "Valor de x usando *p: " << *p << std::endl; // Muestra 10
*p = 20; // Cambia el valor de x a través del puntero
std::cout << "Nuevo valor de x: " << x << std::endl; // Muestra 20
¿Por qué aprender punteros?
  • Permiten modificar variables fuera de la función donde se declaran (paso por referencia).
  • Son esenciales para trabajar con arreglos dinámicos y estructuras de datos complejas.
  • Ayudan a optimizar el uso de memoria y recursos.
¡Cuidado! Usar punteros incorrectamente puede causar errores difíciles de detectar, como acceder a memoria no válida (punteros "colgantes").

Estructuras: Agrupa tus datos

Una estructura (struct) te permite agrupar diferentes tipos de datos bajo un solo nombre. Es como crear tu propio tipo de dato personalizado, donde cada campo puede ser de un tipo distinto.

¿Cómo funciona el ejemplo?
  • Se define una estructura llamada Persona con dos campos: edad (entero) y nombre (cadena de texto).
  • Se crea una variable alumno de tipo Persona.
  • Se asignan valores a los campos de alumno usando el operador punto (.).
  • Se accede a los datos de la estructura igual que a los de una variable normal, usando el punto.
struct Persona {
    int edad;
    std::string nombre;
};

Persona alumno; // Declara una variable de tipo Persona
alumno.edad = 18; // Asigna 18 a la edad
alumno.nombre = "Ana"; // Asigna "Ana" al nombre
std::cout << "Nombre: " << alumno.nombre << ", Edad: " << alumno.edad << std::endl;
¿Para qué sirven?
  • Organizar datos relacionados (por ejemplo, los datos de un estudiante, producto, libro, etc.).
  • Facilitar el manejo de colecciones de objetos complejos (arreglos de estructuras).
Tip: Puedes crear arreglos de estructuras, por ejemplo: Persona grupo[30];

Clases y Programación Orientada a Objetos

Una clase es una extensión de las estructuras: además de agrupar datos, también puede tener funciones (métodos) que operan sobre esos datos. Es la base de la Programación Orientada a Objetos (POO) en C++.

¿Cómo funciona el ejemplo?
  • Se define una clase Animal con un atributo público nombre y un método público hablar().
  • El método hablar() imprime un mensaje usando el nombre del animal.
  • Se crea un objeto perro de tipo Animal, se le asigna un nombre y se llama a su método hablar().
  • El acceso public: permite que los atributos y métodos sean accesibles desde fuera de la clase.
class Animal {
public:
    std::string nombre;
    void hablar() {
        std::cout << "Hola, soy " << nombre << std::endl;
    }
};

Animal perro; // Crea un objeto de tipo Animal
perro.nombre = "Firulais";
perro.hablar(); // Llama al método hablar()
¿Por qué usar clases?
  • Permiten modelar objetos del mundo real con datos (atributos) y comportamientos (métodos).
  • Facilitan la reutilización y organización del código.
  • Permiten usar conceptos avanzados como herencia y polimorfismo.
Tip: Puedes crear múltiples objetos de la misma clase, cada uno con sus propios datos.

Manejo de Archivos en C++

Leer y escribir archivos permite que tu programa guarde información de manera permanente o procese datos externos. En C++ se usan las bibliotecas <fstream> para esto.

¿Cómo funciona el ejemplo?
  • #include <fstream> incluye las clases para manejo de archivos.
  • std::ofstream archivo("datos.txt"); crea (o abre) un archivo para escritura.
  • archivo << "Hola archivo!"; escribe texto en el archivo.
  • archivo.close(); cierra el archivo, asegurando que los datos se guarden correctamente.
  • Para leer, se usa std::ifstream y el método getline para leer línea por línea.
#include <fstream>
#include <iostream>
#include <string>

int main() {
    std::ofstream archivo("datos.txt"); // Abre archivo para escribir
    archivo << "Hola archivo!" << std::endl;
    archivo.close();

    std::ifstream entrada("datos.txt"); // Abre archivo para leer
    std::string linea;
    if (entrada.is_open()) {
        while (getline(entrada, linea)) {
            std::cout << "Leído: " << linea << std::endl;
        }
        entrada.close();
    } else {
        std::cout << "No se pudo abrir el archivo." << std::endl;
    }
    return 0;
}
¿Qué debes recordar?
  • Siempre cierra los archivos después de usarlos.
  • Verifica si el archivo se abrió correctamente antes de leer o escribir.
  • Los archivos se guardan en la misma carpeta que tu programa, a menos que indiques otra ruta.

Manejo de Excepciones

Las excepciones permiten manejar errores de forma controlada. Si ocurre un problema (por ejemplo, división por cero o archivo no encontrado), puedes "lanzar" una excepción y "atraparla" para evitar que el programa se cierre inesperadamente.

¿Cómo funciona el ejemplo?
  • El bloque try contiene el código que podría causar un error.
  • Si ocurre un problema, throw lanza una excepción (en este caso, un mensaje de texto).
  • El bloque catch recibe la excepción y ejecuta el código para manejar el error (por ejemplo, mostrar un mensaje).
  • El programa continúa después del manejo de la excepción.
try {
    int a = 10, b = 0;
    if (b == 0)
        throw "Error: división por cero";
    std::cout << (a / b) << std::endl;
} catch (const char* msg) {
    std::cout << "Excepción capturada: " << msg << std::endl;
}
std::cout << "El programa sigue funcionando." << std::endl;
¿Cuándo usar excepciones?
  • Cuando hay operaciones que pueden fallar (archivos, divisiones, memoria dinámica, etc.).
  • Para mostrar mensajes claros al usuario y evitar que el programa termine abruptamente.
Tip: Puedes lanzar y capturar diferentes tipos de excepciones (int, string, objetos, etc.).

Plantillas (Templates)

Las plantillas permiten escribir funciones o clases que funcionan con cualquier tipo de dato, sin duplicar código. Son la base de la programación genérica en C++.

¿Cómo funciona el ejemplo?
  • template <typename T> indica que la función sumar puede recibir cualquier tipo de dato (T).
  • Cuando llamas a sumar(3, 4), T es int. Si llamas a sumar(2.5, 1.2), T es double.
  • El compilador genera automáticamente la versión adecuada de la función según los tipos de datos usados.
template <typename T>
T sumar(T a, T b) {
    return a + b;
}

std::cout << sumar(3, 4) << std::endl; // Suma enteros: 7
std::cout << sumar(2.5, 1.2) << std::endl; // Suma doubles: 3.7
¿Por qué usar plantillas?
  • Evitas repetir el mismo código para diferentes tipos de datos.
  • Son la base de las estructuras y algoritmos genéricos de la STL.

STL: La Biblioteca Estándar de Plantillas

La STL (Standard Template Library) es una colección de clases y algoritmos genéricos listos para usar, como vector, list, map, set, y muchos más. Facilita el trabajo con estructuras de datos y algoritmos eficientes.

¿Cómo funciona el ejemplo?
  • #include <vector> permite usar la clase std::vector, un arreglo dinámico que puede crecer o reducirse en tiempo de ejecución.
  • std::vector<int> numeros = {1, 2, 3}; crea un vector de enteros con tres elementos.
  • numeros.push_back(4); agrega el número 4 al final del vector.
  • Puedes acceder a los elementos igual que en un arreglo: numeros[0] es 1.
#include <vector>
std::vector<int> numeros = {1, 2, 3}; // Vector con 3 elementos
numeros.push_back(4); // Ahora tiene 4 elementos: 1, 2, 3, 4
std::cout << "Primer elemento: " << numeros[0] << std::endl;
¿Por qué usar STL?
  • Te ahorra tiempo: no necesitas programar tus propias estructuras de datos.
  • Son rápidas, seguras y probadas.
  • Incluyen algoritmos útiles como ordenación, búsqueda, conteo, etc.

Algoritmos en STL

La STL incluye algoritmos listos para usar, como sort (ordenar), find (buscar), count (contar), entre otros. Funcionan con los contenedores de la STL (como vector).

¿Cómo funciona el ejemplo?
  • #include <algorithm> permite usar algoritmos estándar.
  • std::sort(v.begin(), v.end()); ordena el vector v de menor a mayor.
  • v.begin() y v.end() son iteradores que indican el inicio y el final del vector.
  • Después de ordenar, puedes recorrer el vector para ver los elementos ya ordenados.
#include <vector>
#include <algorithm>
#include <iostream>

std::vector<int> v = {3, 1, 4};
std::sort(v.begin(), v.end()); // Ordena: 1, 3, 4
for(int n : v) {
    std::cout << n << " ";
}
std::cout << std::endl;
Tip: Consulta la documentación de algoritmos STL para descubrir todo lo que puedes hacer.
